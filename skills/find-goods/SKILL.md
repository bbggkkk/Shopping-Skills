# Find Goods

인터넷에서 요구받은 상품이 존재하는지 찾고, 유효성을 검증하는 파이프라인 스킬입니다.

## 전체 파이프라인

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    FIND-GOODS PIPELINE                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  [Step 1: 요구사항 분석]                                                │
│       │                                                                 │
│       ▼                                                                 │
│  사용자 입력 → 키워드 추출 → 카테고리 분류 → 검증 기준 설정           │
│                                                                         │
│  [Step 2: 검색 실행]                                                    │
│       │                                                                 │
│       ▼                                                                 │
│  웹 검색 → 쇼핑몰 검색 → 가격 비교 → 결과 통합                        │
│                                                                         │
│  [Step 3: 결과 수집]                                                    │
│       │                                                                 │
│       ▼                                                                 │
│  중복 제거 → 신뢰도 평가 → 기본 필터링 → 상품 목록                     │
│                                                                         │
│  [Step 4: 상품 검증]                                                    │
│       │                                                                 │
│       ▼                                                                 │
│  재고 확인 → 사양 검증 → 가격 검증 → 검증 결과                        │
│                                                                         │
│  [Step 5: 리포트 생성]                                                  │
│       │                                                                 │
│       ▼                                                                 │
│  결과 요약 → 대안 제안 → 최종 리포트                                   │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 파이프라인 단계

| 단계 | 파일 | 설명 |
|------|------|------|
| 1 | `pipeline/01-requirement-parsing.md` | 요구사항 분석 및 검색 키워드 생성 |
| 2 | `pipeline/02-search-execution.md` | 3단계 검색 실행 (웹→쇼핑몰→가격비교) |
| 3 | `pipeline/03-result-collection.md` | 검색 결과 통합 및 중복 제거 |
| 4 | `pipeline/04-product-validation.md` | 재고/사양/가격 유효성 검증 |
| 5 | `pipeline/05-report-generation.md` | 리포트 생성 및 대안 제안 |

## When to Use

- 사용자가 특정 상품을 찾고 있을 때
- 상품의 재고/품절 여부를 확인할 때
- 상품의 사양/스펙이 요구사항과 일치하는지 확인할 때
- 가격 비교가 필요할 때
- 대안 상품을 찾아야 할 때

## 실행 규칙

1. **단계 순차 실행** — 각 단계는 이전 단계의 출력이 필요
2. **에러 시 재시도** — 네트워크 오류 시 최대 3회 재시도
3. **중간 결과 저장** — 각 단계 완료 시 결과 저장
4. **사용자 확인** — 최종 결과는 사용자에게 제시 후 확인
5. **대안 제안** — 검증 실패 시 유사 상품 제안

## 에러 핸들링

| 상황 | 처리 방법 |
|------|-----------|
| **검색 실패** | 해당 소스 건너뛰고 다음 소스로 진행 |
| **네트워크 오류** | 최대 3회 재시도, 실패 시 로그 기록 |
| **결과 없음** | 대안 검색 키워드로 재검색 제안 |
| **검증 실패** | 실패 사유 명시 및 대안 상품 제안 |
