# Step 5: 리포트 생성

## 개요

검증된 상품 목록을 바탕으로 최종 리포트를 생성합니다. 검증 결과 요약, 대안 상품 제안, 구매 가이드를 포함합니다.

## 리포트 구성

### 1. 검색 결과 요약

| 항목 | 내용 |
|------|------|
| **검색 키워드** | 사용된 검색 키워드 |
| **총 검색 결과** | 수집된 상품 수 |
| **검증 결과** | 통과/조건부통과/실패 건수 |
| **가격 범위** | 최저가 ~ 최고가 |

### 2. 상품 상세 정보

각 검증된 상품에 대해 다음 정보를 표시합니다:

```markdown
### [상품명] - [검증결과]

- **가격**: [가격]원
- **판매처**: [판매처명]
- **재고 상태**: [재고상태]
- **주요 사양**: [핵심 사양 나열]
- **평점**: [별점] ([리뷰수]건)
- **검증 결과**: [PASS/CONDITIONAL PASS/FAIL]

[상세 검증 결과]
```

### 3. 검증 결과 표시

| 검증 항목 | 결과 | 상세 |
|-----------|------|------|
| 재고/품절 여부 | ✅/⚠️/❌ | 상세 설명 |
| 사양/스펙 일치 | ✅/⚠️/❌ | 일치/불일치 사양 |
| 제품 특성 일치 | ✅/⚠️/❌ | 카테고리/브랜드/모델 |
| 가격 정합성 | ✅/⚠️/❌ | 시장 평균 대비 비교 |

### 4. 대안 상품 제안

검증 실패한 상품이나 대안이 필요한 경우 유사 상품을 제안합니다:

| # | 상품명 | 가격 | 차이점 | 추천 이유 |
|---|--------|------|--------|-----------|
| 1 | 대안상품명 | 가격 | 차이점 설명 | 추천 사유 |

### 5. 구매 가이드

최종 구매를 위한 가이드를 제공합니다:

- **최적의 선택**: 가장 높은 검증 점수를 받은 상품
- **가성비 선택**: 가격 대비 성능이 우수한 상품
- **프리미엄 선택**: 최고 사양의 상품

## 리포트 형식

```markdown
# 상품 검색 리포트: [상품명]

## 검색 요약
- **검색 키워드**: [키워드]
- **총 검색 결과**: [건수]건
- **검증 결과**: 통과 [건수]건, 조건부통과 [건수]건, 실패 [건수]건
- **가격 범위**: [최저가]원 ~ [최고가]원
- **검색 시점**: [날짜]

---

## 검증된 상품

### 1. [상품명] - ✅ PASS

**기본 정보**
- **브랜드**: [브랜드]
- **모델번호**: [모델]
- **가격**: [가격]원
- **판매처**: [판매처]
- **재고 상태**: 재고 있음
- **평점**: ⭐ [별점] ([리뷰수]건)

**주요 사양**
- CPU: [사양]
- RAM: [사양]
- 저장용량: [사양]
- 화면: [사양]

**검증 결과**
| 항목 | 결과 | 상세 |
|------|------|------|
| 재고/품절 | ✅ PASS | 재고 있음 |
| 사양/스펙 | ✅ EXACT MATCH | 모든 핵심 사양 일치 |
| 제품 특성 | ✅ PASS | 카테고리/브랜드/모델 일치 |
| 가격 정합성 | ✅ PASS | 시장 평균 대비 -2.6% |

**종합 점수**: 92/100
```

## 규칙

1. **검증 결과 반영** — 검증 결과를 정확하게 반영
2. **대안 제안** — 실패 시 최소 1개 이상 대안 제안
3. **가격 정보 명시** — 검색 시점 기준임을 명시
4. **실시간 아님** — 재고/가격 정보가 실시간이 아님을 안내
