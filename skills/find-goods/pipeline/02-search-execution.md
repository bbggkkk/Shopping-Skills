# Step 2: 검색 실행

## 개요

생성된 검색 키워드를 사용하여 3단계 검색을 실행합니다.

## 검색 단계

| 단계 | 소스 | 도구 | 우선순위 |
|------|------|------|----------|
| **1차** | 웹 검색 엔진 | `websearch` | 높음 |
| **2차** | 온라인 쇼핑몰 | `websearch` + `webfetch` | 중간 |
| **3차** | 가격 비교 사이트 | `websearch` + `webfetch` | 낮음 |

## 1차 검색: 웹 검색 엔진

### 검색 전략

```
검색 쿼리: [상품명] + [브랜드] + [사양]
예시: "갤럭시북 프로 360 삼성 15인치 i7 16GB"
```

### 검색 대상

| 검색엔진 | 특징 | 사용 시점 |
|----------|------|-----------|
| **Google** | 가장 광범위한 결과 | 기본 검색 |
| **Bing** | 이미지/쇼핑 결과 강화 | 이미지 확인 필요 시 |
| **Naver** | 한국어 콘텐츠 특화 | 한국 상품 검색 시 |

### 수집 정보

- 상품명, 브랜드, 모델
- 가격 정보
- 판매처 URL
- 리뷰/평점 요약

## 2차 검색: 온라인 쇼핑몰

### 검색 대상 쇼핑몰

| 쇼핑몰 | 특징 | 검색 방법 |
|--------|------|-----------|
| **쿠팡** | 빠른 배송, 다양한 상품 | "site:coupang.com [상품명]" |
| **네이버 쇼링** | 가격 비교, 리뷰 | "site:shopping.naver.com [상품명]" |
| **11번가** | 다양한 할인 | "site:11st.co.kr [상품명]" |
| **G마켓** | 중고/리퍼 가능 | "site:gmarket.co.kr [상품명]" |
| **Amazon** | 해외 직구 | "site:amazon.com [상품명]" |

### 수집 정보

- 정확한 가격
- 재고 상태
- 배송 정보
- 판매자 정보
- 상세 스펙

## 3차 검색: 가격 비교 사이트

### 검색 대상

| 사이트 | 특징 | 사용 시점 |
|--------|------|-----------|
| **다나와** | 전자제품 가격 비교 | 전자제품 검색 시 |
| **네이버 가격비교** | 범용 가격 비교 | 일반 상품 검색 시 |
| **에누리** | 실시간 가격 추적 | 가격 변동 확인 시 |

### 수집 정보

- 최저가 정보
- 가격 변동 이력
- 판매처별 가격 비교
- 리뷰/평점 종합

## 검색 실행 규칙

1. **단계별 실행** — 1차 → 2차 → 3차 순서로 실행
2. **결과 통합** — 각 단계의 결과를 통합하여 처리
3. **에러 시 건너뛰기** — 검색 실패 시 다음 단계로 진행
4. **중복 제거** — 동일 상품은 하나로 통합

## 에러 핸들링

| 상황 | 처리 방법 |
|------|-----------|
| **검색엔진 차단** | 다른 검색엔진으로 변경 |
| **쇼핑몰 접속 불가** | 해당 쇼핑몰 건너뛰기 |
| **결과 없음** | 키워드 수정 제안 |
| **네트워크 오류** | 최대 3회 재시도 |
